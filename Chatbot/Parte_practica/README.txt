Prueba Tecnica por María Alejandra Estupiñán Murillo

* Para ejecutar el codigo,simplemente abre el archivo index.html que se encuentra en 
la carpeta chatbot en el github

* El proyecto consiste en tres partes,cada una en archivos separados dentro de la carpeta 
principal, es decir en la carpeta se encuentra un archivo html que tiene toda la estructura del chatbot, 
el archivo CSS define los estilos visuales del chatbot para crear una interfaz atractiva y coherente,
el archivo Javascript controla la interacción con el usuario, gestionando la logica de funcionamiento
del chatbot.Ademas, hay una carpeta adicional que almacena las imagenes utilizadas en el proyecto.
